# Modern Quiz Web Application

## Overview

This is a modern interactive quiz web application developed using HTML, CSS, and JavaScript. The application allows users to enter their name, answer multiple-choice questions, navigate between questions, and view their final score at the end of the quiz.

## Features

* User name entry screen
* Multiple-choice quiz
* Answer selection highlighting
* Previous and Next navigation
* Mandatory answer selection
* Score calculation
* Thank you screen
* Restart quiz option
* Responsive modern UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download the project
2. Open index.html in a web browser
3. Enter your name and start the quiz
